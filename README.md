
# Compose Markdown

Compose Markdown is a jetpack compose extensible markdown rendering library which uses CommonMark (java , markdown parsing library) to render composables
  
## Usage
  
For preview use `MarkdownPreview`
  
```kotlin
MarkdownPreview(
    markdown = text
)
```
